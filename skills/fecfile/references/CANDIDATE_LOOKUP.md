# Candidate-to-Committee Lookup via FEC Bulk DataUse this workflow when `search_committees` returns no results for a person's name.The MCP tool searches committee names only — it won't find "Friends of Jane Smith" fromthe query "Jane Smith". FEC bulk data links candidates to their committees directly.---## Why Bulk Data?The FEC maintains separate registries for **candidates** (Form 2) and **committees** (Form 1).A candidate's principal campaign committee is registered under its own name (e.g.,"Platner for Congress"), not the candidate's name. The bulk candidate file mapscandidate names → candidate IDs → principal committee IDs, bridging the gap.---## Key Bulk Data FilesAll files are available at `https://www.fec.gov/data/bulk-downloads/` and updated nightly.| File | URL | Description ||------|-----|-------------|| Candidate master | `https://www.fec.gov/files/bulk-downloads/2026/cn26.zip` | All candidates, current cycle || Candidate–committee linkage | `https://www.fec.gov/files/bulk-downloads/2026/ccl26.zip` | Maps candidate IDs to committee IDs || Committee master | `https://www.fec.gov/files/bulk-downloads/2026/cm26.zip` | All committees (for cross-reference) |Replace `2026` with the relevant two-year cycle (e.g., `2024`, `2022`).---## Candidate Master File (`cn26.zip` → `cn.txt`)Pipe-delimited, no header row. Columns in order:| # | Field | Notes ||---|-------|-------|| 0 | `CAND_ID` | e.g., `P00003392` || 1 | `CAND_NAME` | Last, First format: `SMITH, JANE` || 2 | `CAND_PTY_AFFILIATION` | `DEM`, `REP`, `IND`, etc. || 3 | `CAND_ELECTION_YR` | Most recent election year || 4 | `CAND_OFFICE_ST` | Two-letter state (blank for President) || 5 | `CAND_OFFICE` | `P`=President, `S`=Senate, `H`=House || 6 | `CAND_OFFICE_DISTRICT` | House district number (00 = Senate/President) || 7 | `CAND_ICI` | Incumbent/Challenger/Open: `I`, `C`, `O` || 8 | `CAND_STATUS` | `C`=Statutory candidate, `F`=Statutory candidate for future election, `N`=Not yet a statutory candidate, `P`=Statutory candidate in prior cycle || 9 | `CAND_PCC` | Principal campaign committee ID || 10 | `CAND_ST1` | Mailing address street 1 || 11 | `CAND_ST2` | Mailing address street 2 || 12 | `CAND_CITY` | City || 13 | `CAND_ST` | State || 14 | `CAND_ZIP` | ZIP code |**Key fields for lookup**: `CAND_NAME` (col 1), `CAND_ID` (col 0), `CAND_PCC` (col 9).---## Candidate–Committee Linkage File (`ccl26.zip` → `ccl.txt`)Pipe-delimited, no header row. Columns in order:| # | Field | Notes ||---|-------|-------|| 0 | `CAND_ID` | Links to candidate master || 1 | `CAND_ELECTION_YR` | Election year for this linkage || 2 | `FEC_ELECTION_YR` | FEC two-year cycle || 3 | `CMTE_ID` | Committee ID (use with `get_filings`) || 4 | `CMTE_TP` | Committee type || 5 | `CMTE_DSGN` | `P`=Principal, `A`=Authorized, `J`=Joint fundraiser || 6 | `LINKAGE_ID` | Internal FEC linkage ID |Use `CMTE_DSGN = 'P'` to find the principal campaign committee.---## Lookup Workflow### Step 1 — Download and extract the candidate file```bashcurl -sL https://www.fec.gov/files/bulk-downloads/2026/cn26.zip -o /tmp/cn26.zipunzip -p /tmp/cn26.zip cn.txt > /tmp/cn.txt```### Step 2 — Search by candidate name (case-insensitive)Names are stored as `LAST, FIRST` in uppercase.```bashgrep -i "PLATNER" /tmp/cn.txt```Each matching line is pipe-delimited. Note the `CAND_ID` (col 0) and `CAND_PCC` (col 9).### Step 3 — If CAND_PCC is populated, use it directly`CAND_PCC` is the principal committee ID. Pass it straight to `get_filings`:```python# Use the MCP tool:get_filings(committee_id="C00XXXXXX")```### Step 4 — If CAND_PCC is blank, use the linkage file```bashcurl -sL https://www.fec.gov/files/bulk-downloads/2026/ccl26.zip -o /tmp/ccl26.zipunzip -p /tmp/ccl26.zip ccl.txt > /tmp/ccl.txtgrep "CAND_ID_HERE" /tmp/ccl.txt | awk -F'|' '$6=="P" {print $4}'```### Step 5 — Cross-reference committee name (optional)```bashcurl -sL https://www.fec.gov/files/bulk-downloads/2026/cm26.zip -o /tmp/cm26.zipunzip -p /tmp/cm26.zip cm.txt > /tmp/cm.txtgrep "COMMITTEE_ID_HERE" /tmp/cm.txt```---## Multi-Cycle SearchCandidates who ran in prior cycles may not appear in `cn26.zip`. Try adjacent cycles:```bashfor year in 2026 2024 2022 2020; do  curl -sL "https://www.fec.gov/files/bulk-downloads/${year}/cn${year:2}.zip" -o /tmp/cn${year}.zip  unzip -p /tmp/cn${year}.zip cn.txt 2>/dev/null | grep -i "PLATNER" && echo "(found in ${year})"done```---## Notes- Bulk files are regenerated nightly — cached copies may be stale for very recent registrations.- `CAND_PCC` may be blank for candidates who haven't yet designated a principal committee.- Candidates can have multiple authorized committees; the linkage file lists all of them.- For state/local races, candidates file with state agencies, not the FEC — bulk data only covers federal candidates (President, Senate, House).
